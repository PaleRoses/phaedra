# Lua Reference

Phaedra provides Lua 5.4 as a configuration language. This section documents
the various lua functions and types that are provided to the configuration
file. These are provided by the `phaedra` module that must be imported into
your configuration file:

```lua
local phaedra = require 'phaedra'
local config = {}
config.font = phaedra.font 'JetBrains Mono'
return config
```
